# 🌡️ Weather Stats Challenge (Análise Climática)

Sistema em C desenvolvido para análise de dados climáticos. O algoritmo processa temperaturas para identificar os picos de calor e frio,
além de calcular e comparar a média térmica de cada cidade com a média geral.

## 🚀 Funcionalidades

**Entrada Dinâmica:** O usuário define a quantidade de cidades e a quantidade de dias analisados logo no início da execução.
* **Mapeamento de Extremos:** O sistema varre os dados para encontrar a maior e a menor temperatura registrada, apontando exatamente o dia e a cidade dessas ocorrências.
* **Cálculo de Médias:** Gera a temperatura média individual de cada cidade e a média geral englobando todos os dados informados.
* **Gerenciamento de Memória:** Implementação de matrizes dinâmicas utilizando ponteiros e a função `malloc` para alocação de memória sob demanda, garantindo eficiência.

## 🛠️ Tecnologias e Conceitos

* **Linguagem:** C
* **Conceitos Aplicados:**
  * Alocação Dinâmica de Memória (Ponteiros para Ponteiros `**`).
  * Manipulação de Matrizes.
  * Estruturas de repetição aninhadas (`for`).
  * Lógica de programação para análise de dados.

## 💻 Como Executar

Para rodar esse projeto no seu computador, você precisará de um compilador C (como o GCC) instalado.
